# Задание 1
# Генератор заданий по машинному представлению вещественных чисел

## Описание
- Программа генерирует N вариантов с K задачами (вещественные числа) и файлы проверки с машинным представлением (32/64 бит) и ошибкой округления.
- Язык: C (один файл first.c).

## Сборка
```bash
gcc -Wall -Wextra -o output/first first.c -lm
```

Формат входного файла
- Файл: ../input.txt (относительно рабочего каталога запуска программы; см. примечание о путях ниже)
- Порядок строк/значений:
  1. N — количество вариантов (int)
  2. K — количество заданий на вариант (int)
  3. bits_type — разрядность: 32 или 64 (int)
  4. range_a range_b — диапазон вещественных чисел (float float)
  5. precision — количество знаков после запятой при выводе (int)

## Пример input.txt
```
5
10
32
-10.0 10.0
3
```

## Запуск
- Запустить собранный бинарник в каталоге проекта:
```bash
cd output && ./first
```

## Выходные файлы
- Программа создаёт Markdown-файлы с заданиями и проверками:
  - tasks/variant_X_task.md — таблица заданий (N, вещественное число)
  - checks/variant_X_check.md — таблица проверки: число, машинное представление, ошибка
- Если директории отсутствуют — программа пытается создать их (mkdir).

## Пояснения
- Для 32-бит: число округляется до float, затем формируется битовое представление, вычисляется погрешность относительно исходного double.
- Для 64-бит: используется double напрямую; машинное представление формируется из битов double.
- Биты извлекаются через приведение указателей к целочисленным типам и побитовые операции.

### Пример вывода
variant_X_task.md
```markdown

| N | Вещ число |

| 1 | 86.95823 |
| 2 | 99.52941 |
| 3 | 66.21308 |
| 4 | 17.75227 |
```
